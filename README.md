# Jogo Da Forca

![](https://imgur.com/L8amxUG.gif)

## Introdução

Este é o jogo da forca e seu objetivo é adivinhar a palavra secreta antes de ser enforcado por seus erros.

## Funcionalidades

- **Categorias**: É possível escolher entre 4 categorias de palavras diferentes.
- **Palavras Aleatórias**: Para cada partida é sorteada uma palavra dentre as várias da categoria escolhida.
- **Sem Repetição de Letras**: Não é possível chutar a mesma letra mais de uma vez.
- **Possibilidade de Chutar uma Palavra**: É possível chutar uma palavra inteira ao invés de letra por letra. Se errar a palavra, conta como apenas 1 erro.
- **Desenho da Forca**: A cada chute errado, o desenho da forca é atualizado, o que deixa o jogo mais interessante visualmente.
- **Repetição do Jogo**: É possível repetir o jogo quantas vezes quiser, selecionando categorias diferentes a cada vez.
